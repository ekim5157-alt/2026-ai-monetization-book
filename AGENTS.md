# AGENTS.md

이 저장소는 『2026 AI 수익화 전체 지도』 제작용입니다.

모든 작업은 README.md의 디렉터리와 파일명 규칙을 따릅니다. 한 번에 한 모델만 최종 편집합니다.

고정 작업 순서:

`business → research → draft → review → final`

단계별 선행 입력:

- `business`는 해당 모델의 실제 병목, 수익 구조, 중단 기준과 공식 검증이 필요한 조사 체크리스트를 작성합니다.
- `research`는 먼저 `handoff/business/model-XX.md`를 읽고 그 주장과 체크리스트를 공식자료로 검증합니다.
- `draft`는 `handoff/business/model-XX.md`와 `handoff/research/model-XX.md`를 모두 읽고 초보자 실행 원고를 작성합니다.
- `review`는 business, research, draft 세 파일을 함께 읽고 품질을 검증합니다.
- `final`은 네 개의 handoff 파일을 통합하고 검증 지적을 반영해 승인본을 작성합니다.

승인 전 결과는 `sanity-packages/`에 넣지 않습니다. 모든 작업은 채팅 출력으로 끝내지 말고 지정된 GitHub 경로에 저장하고 커밋합니다.
