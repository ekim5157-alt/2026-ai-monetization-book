# 공통 편집 규칙

- 유투브를 제외하고 해외시장, 외국인, 영어 번역을 통한 현실적인 제약을 고려하거나, 그 방향의 확장은 상정하지 않는다.
- 유투브, 인스타그램, 스레드, 틱톡, 블로그, 인포크링크를 제외한 플랫폼은 한국 플랫폼만을 다룬다.
- 구글 검색, 네이버 검색 등을 통해 한국에서 각 사업모델을 위해 많이 쓰이는 한국 플랫폼이 무엇인지 조사한다.
- 플랫폼, 결제, 정산, 세금, 가격, 광고, 구매 습관과 실행 경로는 구체적으로 쓴다.
- 외국인은 이 책의 독자가 아니다.
- 저장 전 `python scripts/style_guard.py`를 실행한다.
- 검사 실패 시 완료 처리, 다음 단계 전달, final 승인, 패키지 생성을 중단한다.
- 미국식 수익모델은 실제 구매자, 유입, 결제, 정산, 비용과 입금 경로가 확인될 때만 채택한다.
- 긴 자료는 `관찰 → 숨은 변수 → 작동 메커니즘 → 실행 판단 → 실패·전환 신호`로 재구성한다.
- 일반론으로 평탄화하지 않는다.
- 이북에서 초보자가 실제로 수익화하는 경로: 브런치스토리, 크몽, 탈잉, 클래스101, 노션 템플릿 마켓류, 교보문고·YES24·알라딘 전자책 유통, 브런치북 출판 프로젝트
- 이북은 개인 홈페이지나 개인 카카오채널에 올리고 스레드에 광고하는 일이 많으므로 이를 스레드에서도 다루도 이북에서도 다룬다.
- 원소스멀티유즈 관점에서 짧은 영상을 유투브, 인스타그램, 틱톡, X(트위터), 페이스북에 올리는 것을 각 사업모델에서 다뤄야 한다.
- 각각의 사업모델의 채널은 독립된 것이 아니고 유기적으로 연결되며 새로운 방법이 생길 수 있다.
- 기존에 효과가 없다고 생각하는 인스타그램 단순 사진이나, 당근에 몇줄 안되는 글이 큰 수익모델이 되는 것과 같이 현재 방법들이 전부가 아니라는 관점.

## 사업모델

### 21. 당근

당근을 단순 지역 광고나 중고거래 모델로 쓰지 않는다. 커뮤니티, 선생님 프로필, 비즈니스 프로필, 피드 광고, 개인샵 링크, 팔로우, 이웃 관계, 후기와 평가가 한 앱 안에서 섞여 작동하는 신뢰 기반 융합 서비스 모델로 설명한다.

공통 전환 흐름:

`피드·커뮤니티 콘텐츠 → 선생님·비즈니스 프로필 → 후기·평가·이웃·팔로우 축적 → 강의·어필리에이트·개인샵·직접 판매 전환`

초기 접점은 생활권에서 생길 수 있지만 사업성의 핵심 단위는 거리보다 신뢰다. 후기, 평가, 반복 노출, 이웃 관계, 팔로우, 거래·참여 이력이 전환을 만든다.

당근은 다음 세 하위 구조를 하나의 모델 안에서 함께 다룬다.

#### 당근 커뮤니티(교육)
다룰 범위:
- 동네 모임
- 소규모 클래스
- 오프라인 강의
- 취미 수업
- AI 활용 교육
- 스마트폰 활용 교육
- 시니어 교육
- 선생님·운영자 프로필

수익 흐름:

`콘텐츠·커뮤니티 → 선생님 프로필 신뢰 → 모임 → 강의 → 컨설팅`

주 수익은 교육과 상담이다. 자기 상품이나 개인샵 링크는 보조 전환으로 다룬다. 조회수보다 모임 신청률, 실제 참석률, 재참여율, 강의 전환율, 상담 전환율을 본다.

#### 당근 광고(어필리에이트)
다룰 범위:
- 피드형 게시물
- 커뮤니티 게시물
- 제품 추천
- 생활 정보형 콘텐츠
- 선생님·운영자 프로필
- 비즈니스 프로필
- 어필리에이트 링크
- 개인샵 링크
- 프로필·댓글·문의로 이어지는 전환 경로

수익 흐름:

`피드·커뮤니티 노출 → 프로필 신뢰 → 링크·문의 → 어필리에이트·자기 상품 전환 → 수익`

핵심은 별도 광고 지면만 사는 것이 아니라 일반 피드와 커뮤니티 문맥에 자연스럽게 섞이는 콘텐츠에서 신뢰를 만들고 프로필·개인샵으로 이동시켜 전환을 만드는 구조다. 프로필 방문률, 팔로우 증가, 링크 이동률, 문의율, 구매 전환율, 건당 수익을 본다.

#### 당근 비즈니스(개인샵/판매전환)
다룰 범위:
- 비즈니스 프로필
- 개인샵
- 자기 상품
- 중고 판매
- 리셀
- 재고 처리
- 소규모 커머스
- 직접 판매

수익 흐름:

`상품·콘텐츠 → 비즈니스 프로필·개인샵 → 신뢰 확인 → 직접 판매 → 현금흐름`

핵심은 후기, 평가, 팔로우, 거래 이력으로 판매 마찰을 낮추고 반복 판매와 빠른 현금 회수를 만드는 구조다. 매입가, 판매가, 회전일수, 프로필 방문률, 반복 구매율, 이동·응대 시간, 반품·분쟁 비용을 함께 본다.

세 하위 구조는 서로 분리하지 않는다.
- 커뮤니티와 선생님 프로필은 강의·상담뿐 아니라 어필리에이트와 개인샵 전환에도 기여할 수 있다.
- 피드와 비즈니스 프로필은 어필리에이트와 자기 상품 판매를 동시에 만들 수 있다.
- 판매·거래 이력은 다시 프로필 신뢰와 팔로우를 높여 교육과 광고 전환을 강화할 수 있다.
- 구분 기준은 최종 수익이 교육·상담, 어필리에이트·자기 상품, 직접 판매 중 어디에서 발생하는가이다.

사업성 분석에서 반드시 검증할 것:
- 피드·커뮤니티 게시물이 선생님·비즈니스 프로필과 개인샵 방문으로 실제 연결되는가
- 후기·평가·이웃·팔로우가 신청·클릭·구매 전환에 어떤 영향을 주는가
- 어필리에이트 링크와 개인샵 링크 중 어떤 경로가 더 잘 작동하는가
- 비즈니스 프로필과 개인샵이 단일 게시물보다 반복 판매에 유리한가
- 직거래와 간편 판매가 판매대금 회수 속도를 높이는가
- 반복 게시가 신뢰 축적과 피로도·신고 위험 중 어느 쪽으로 작용하는가
- 어필리에이트 표시와 외부 링크 사용이 플랫폼 정책상 허용되는가
- 콘텐츠 제작, 응대, 이동, 보관, 분쟁 비용을 제하고도 반복 가능한가
