# Research handoff

전체 작업의 두 번째 단계입니다. 공식자료 조사 전에 반드시 같은 모델의 `handoff/business/model-XX.md`를 읽습니다.

파일명: `model-XX.md`

필수 항목:
1. 모델명
2. 공식 확인일
3. 사업성 분석에서 제기한 핵심 주장과 검증 결과
4. 조사 체크리스트 항목별 공식 확인 결과
5. 제품별 가격·무료 한도·상업 이용·수익화 및 지급 조건
6. 공식 출처 목록
7. 확인되지 않은 주장
8. 총괄 편집실 인수인계

가격과 정책은 공식 문서를 우선하고, 사실과 추정을 분리합니다. 사업성 분석의 가설이 틀리거나 과장됐으면 그대로 수정 의견을 남깁니다.

완료 후 `status/model-status.csv`의 해당 모델 `조사완료` 상태를 갱신합니다.
