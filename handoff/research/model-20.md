## 핵심 결론

소형 AI SaaS·GPT·웹앱의 공식 정책상 핵심 병목은 **기능 수가 아니라 반복 사용, 사용자당 매출보다 낮은 API·호스팅·DB·지원 원가, 무료 사용량 통제, 데이터 보존·삭제, 결제·해지·환불과 외부 서비스 장애를 감당하는 운영 구조**입니다.

OpenAI API는 ChatGPT 구독과 별도로 사용량 과금됩니다. 2026년 6월 21일 표준 처리 기준 GPT-5.5는 입력 100만 토큰당 5달러·캐시 입력 0.50달러·출력 30달러, GPT-5.4는 2.50달러·0.25달러·15달러, GPT-5.4 mini는 0.75달러·0.075달러·4.50달러입니다. 웹 검색은 1,000회당 10달러이며 Batch API는 입력·출력 비용을 50% 줄이는 대신 최대 24시간 비동기 처리입니다. 무료 사용자가 늘면 매출보다 API비가 먼저 커질 수 있으므로 호출당 비용이 아니라 사용자당 월 총비용을 계산해야 합니다.

OpenAI API 데이터는 명시적으로 동의하지 않는 한 모델 학습에 사용되지 않습니다. 그러나 기본 악용 모니터링 로그는 최대 30일 보관될 수 있고, Responses API의 상태 저장·Conversations·파일·벡터 저장소 등은 기능별 보존기간이 다릅니다. `학습에 사용되지 않는다`와 `아무 데이터도 저장되지 않는다`를 같은 의미로 설명하면 안 됩니다.

Vercel Hobby는 공식적으로 개인·비상업용입니다. 상업 웹앱은 Pro 이상을 검토해야 하며 Pro는 월 20달러와 추가 사용량, 20달러의 포함 사용 크레딧을 안내합니다. Supabase Free는 두 개의 무료 프로젝트와 500MB 데이터베이스·50,000 MAU·1GB 저장공간 등의 한도를 제공하지만, 실제 서비스에서는 조직 단위 사용량 합산과 초과비·프로젝트별 컴퓨트 비용을 계산해야 합니다.

## 성패를 가르는 것

### 1. 가입보다 반복 사용을 검증할 것

한 번 재미로 사용한 데모와 매주 비용을 내는 제품은 다릅니다. 다음을 별도로 기록합니다.

- 가입 대비 첫 결과 완료율
- 7일·30일 재사용률
- 무료 한도 소진 후 유료 전환율
- 월 구독 유지율과 이탈률
- 사용자당 API·DB·호스팅·지원비
- 환불과 결제 실패 이후 고객당 순수익

재사용률이 낮으면 기능 추가보다 반복해서 돌아올 문제를 다시 정의해야 합니다.

### 2. 무료 사용량을 마케팅 비용으로 계산할 것

무료 체험과 무료 플랜은 비용이 없는 기능이 아닙니다. 각 사용자의 모델 입력·출력, 검색·이미지·음성, 재시도, DB 읽기·쓰기, 파일 저장, 네트워크와 지원비를 운영자가 부담합니다.

무료 한도에는 다음 통제가 필요합니다.

1. 사용자·조직·IP별 호출 제한
2. 고비용 모델과 도구의 일일 한도
3. 긴 파일·긴 컨텍스트·반복 요청 제한
4. 결제 전 서버측 권한 확인
5. 비용 급증 알림과 하드 중지
6. 악용·다중계정·자동 호출 탐지

OpenAI API 월 예산은 집행에 지연이 있을 수 있으므로 초과비가 전혀 발생하지 않는 절대 차단 장치로 단정하지 않습니다.

### 3. 모델 가격만 아니라 전체 단위경제성을 볼 것

API 비용에는 입력과 출력 외에도 캐시, 검색, 이미지·음성, 컨테이너, 파일·벡터 저장, 재시도와 실패 요청이 포함될 수 있습니다. 호스팅에는 함수 실행, 전송량, 빌드, 로그와 모니터링이 들어가며 DB에는 활성 사용자, 저장공간, 네트워크, 백업과 컴퓨트 비용이 들어갑니다.

고객당 월 순이익은 다음처럼 봅니다.

`결제액 - 결제수수료 - 세금·환불 - 모델·도구 호출비 - 호스팅·DB·저장비 - 고객지원·장애대응비`

회원 수나 월 반복매출만으로 제품이 흑자인지 판단하면 안 됩니다.

### 4. 데이터 학습 제외와 보존·삭제를 구분할 것

OpenAI API 공식 문서 기준:

- API 데이터는 명시적으로 옵인하지 않으면 모델 학습에 사용되지 않습니다.
- 기본 악용 모니터링 로그에는 프롬프트·응답이 포함될 수 있고 최대 30일 보관될 수 있습니다.
- Responses API는 기본 또는 `store=true`일 때 응답 상태를 최소 30일 저장할 수 있습니다.
- Conversations·ChatKit threads와 일부 저장형 객체는 삭제할 때까지 보관될 수 있습니다.
- 파일은 직접 삭제하거나 만료 설정을 사용해야 합니다.
- Zero Data Retention은 사전 승인 대상이며 모든 엔드포인트와 기능이 동일하게 지원되는 것은 아닙니다.
- 외부 MCP·결제·분석 서비스로 전달된 데이터는 해당 제3자의 보존정책을 따릅니다.

제품에는 개인정보 최소수집, 보유기간, 사용자 삭제, 계정 탈퇴, 백업 삭제, 관리자 접근기록을 설계해야 합니다.

### 5. 무료 인프라를 상업 운영비 0원으로 보지 않을 것

Vercel Hobby는 개인·비상업용으로 명시되어 있습니다. Pro는 월 20달러와 추가 사용량 구조이며 20달러의 포함 사용 크레딧을 제공합니다. Enterprise에만 99.99% SLA가 표시되어 있으므로 무료·Pro 프로젝트에 동일한 가용성을 약속하면 안 됩니다.

Supabase Free는 두 프로젝트와 다음 공식 한도를 안내합니다.

- egress 5GB
- 데이터베이스 500MB/프로젝트
- 월 활성 사용자 50,000명
- 저장공간 1GB
- Edge Function 호출 500,000회
- Realtime 메시지 200만 건
- 최대 동시 연결 200

무료 한도를 넘긴 실제 서비스에서는 유료 플랜과 초과 사용량이 적용되고, 프로젝트별 컴퓨트·커스텀 도메인처럼 한도 없이 전체 사용량이 과금되는 항목도 있습니다.

### 6. GPT 배포와 자체 SaaS 수익화를 구분할 것

ChatGPT 안의 GPT 또는 앱은 빠른 검증·배포 채널이 될 수 있지만, 모든 제작자에게 적용되는 고정 수익배분율과 지급 자격이 공식적으로 보장된 판매채널로 간주하면 안 됩니다. 수익화가 계정·국가·프로그램 초대에 따라 달라질 수 있으므로, 본문에서는 GPT 자체 보상보다 자체 웹앱의 결제·고객 데이터·원가 구조를 중심으로 설명해야 합니다.

자체 웹앱은 사용자가 직접 결제하지만 인증·결제실패·해지·환불·세금·개인정보·고객지원 책임도 운영자에게 생깁니다.

2. GPT-5.5 표준 가격은 입력 5달러·캐시 입력 0.50달러·출력 30달러/100만 토큰입니다.
3. GPT-5.4는 2.50달러·0.25달러·15달러, GPT-5.4 mini는 0.75달러·0.075달러·4.50달러입니다.
4. 웹 검색은 1,000회당 10달러이며 Batch API는 입력·출력 비용을 50% 낮추고 최대 24시간 처리합니다.
5. API 데이터는 명시적 옵인 없이는 모델 학습에 사용되지 않습니다.
6. 기본 악용 모니터링 로그는 최대 30일 보관될 수 있습니다.
7. Responses API와 상태형 객체·파일은 기능별 보존기간과 삭제 방식이 다릅니다.

12. 결제대행 수수료·지원 국가·정산 일정은 사업자 국가와 계정에 따라 다르므로 실제 계약할 결제대행사의 공식 계정 화면을 확인해야 합니다.

## 추정 또는 편집 판단

다음은 공식 보장 기준이 아닙니다.

- 가입자가 많으면 제품의 시장 적합성이 입증된다는 주장
- 무료 사용자가 유료 고객으로 자연스럽게 전환된다는 주장
- 7일·30일 재사용률의 특정 수치가 성공을 보장한다는 주장
- 소형 모델을 쓰면 항상 품질과 비용이 최적이라는 주장
- Vercel·Supabase 무료 플랜만으로 상업 서비스를 계속 운영할 수 있다는 주장
- GPT를 공개하면 자동으로 수익배분을 받을 수 있다는 주장
- 외부 모델 장애가 드물므로 별도 복구 구조가 필요 없다는 주장

재사용률·사용자당 순이익·월 이탈률을 핵심 지표로 보는 것은 사업성 판단입니다.

## 오늘·3일·7일·30일

- 오늘: 한 사용자가 반복해서 해결할 문제와 기존 처리시간, 예상 결제 가치를 정합니다.
- 3일: 한 입력과 한 결과만 있는 프로토타입에 사용자·조직별 비용 한도를 붙입니다.
- 7일: 실제 사용자의 첫 결과 완료·재사용·오류·지원 요청과 사용자당 API비를 측정합니다.
- 30일: 7일·30일 재사용률, 유료 전환, 이탈, 환불, API·인프라·지원비로 계속할지 결정합니다.

## 위험과 주의사항

1. 무료 사용자의 API 비용을 0원으로 계산하지 않습니다.
2. API 학습 제외를 무보존으로 설명하지 않습니다.
3. 민감정보를 필요 이상 저장하거나 로그에 그대로 남기지 않습니다.
4. 사용자 탈퇴와 데이터 삭제, 결제 해지, 환불 경로를 명확히 제공합니다.
5. 모델 결과를 확정적 전문 판단으로 제시하지 않고 오류 안내와 사람 확인 경로를 둡니다.
6. Vercel Hobby를 상업 운영용 무료 서버로 권하지 않습니다.
7. 인프라 무료 한도와 유료 전환 이후 단가를 함께 계산합니다.
8. 외부 모델·DB·결제 서비스 장애 시 기능 축소·재시도·대체 경로와 상태 안내를 둡니다.
9. 결제액과 환불·차지백·수수료 이후 실제 입금을 분리합니다.

## 총괄이 반드시 확인할 것

- 가입자 수보다 반복 사용·사용자당 비용·이탈률을 중심으로 설명할 것.
- OpenAI API 가격에는 2026년 6월 21일 기준일과 변동 가능성을 붙일 것.
- ChatGPT 구독과 API 사용료를 같은 상품으로 설명하지 말 것.
- 학습 제외와 로그·상태 데이터 보존을 구분할 것.
- Vercel Hobby의 비상업 조건과 Supabase 무료 한도의 운영 한계를 포함할 것.
- GPT 공개를 보장된 수익화 경로로 설명하지 말 것.
- 데모 개발보다 로그인·결제·삭제·환불·모니터링·지원이 운영비의 큰 부분임을 설명할 것.
