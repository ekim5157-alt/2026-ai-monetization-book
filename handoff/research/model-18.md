## 핵심 결론

AI 콘텐츠 제작 대행의 공식 정책상 핵심 병목은 **고객 입력자료를 사용할 권한, AI 출력의 비고유성과 사실오류, 이미지·음악·폰트별 상업 라이선스, 납품·수정·2차 사용 범위를 정하는 계약**입니다.

OpenAI 이용약관상 사용자와 OpenAI 사이에서는 사용자가 입력의 권리를 유지하고 출력물을 소유합니다. 그러나 사용자는 입력자료에 필요한 권리와 허가를 갖춰야 하며, 출력은 다른 이용자의 결과와 유사할 수 있고 정확성이 보장되지 않습니다. 따라서 대행사가 고객에게 `AI 출력은 독점적이며 저작권 문제가 없다`거나 `사실 오류가 없다`고 보장할 수 없습니다.

OpenAI의 ChatGPT Business·Enterprise와 API Platform은 비즈니스 데이터를 기본적으로 모델 학습에 사용하지 않는다고 안내합니다. 반면 개인용 서비스는 별도의 설정과 약관이 적용됩니다. 고객의 비공개 자료를 처리할 때는 개인 계정에 무심코 입력하지 말고 계약에 맞는 비즈니스 제품, 접근권한, 보존·삭제 정책을 확인해야 합니다.

## 성패를 가르는 것

1. 고객의 목표를 납품물 형식·수량·길이·채널·품질기준으로 바꿉니다.
2. 수정 횟수와 범위, 고객 자료 지연, 범위 밖 재작업 비용을 계약에 명시합니다.
3. 고객이 제공한 로고·사진·문서·음성에 필요한 이용권한이 있는지 확인합니다.
4. AI 출력은 사실·브랜드 문체·권리·정책을 사람이 검수합니다.
5. 완성본 사용권, 편집 가능한 원본, 재판매·광고 활용, 포트폴리오 공개를 별도 항목으로 정합니다.

## 공식 확인 사실

### AI 입력과 출력

- OpenAI 이용약관상 사용자는 입력에 대한 권리를 유지하고, 법이 허용하는 범위에서 OpenAI와의 관계상 출력물을 소유합니다.
- 사용자는 입력자료에 필요한 권리·라이선스·허가를 보유해야 합니다.
- 출력은 고유하지 않을 수 있으며 다른 사용자에게 유사한 출력이 제공될 수 있습니다.
- 출력은 부정확할 수 있으므로 사용·공유 전에 정확성과 적합성을 사람이 검토해야 합니다.
- AI가 만든 결과를 사람이 만들었다고 허위 표시해서는 안 됩니다.
- ChatGPT Business·Enterprise와 API Platform의 비즈니스 데이터는 기본적으로 모델 학습에 사용되지 않습니다.

### CapCut 소재

- CapCut 내부 소재는 비상업 전용 소재와 제한적 상업 이용이 가능한 Dual Use Materials로 구분됩니다.
- 상업 이용 가능 소재는 `commercial use` 또는 이에 준하는 표시로 식별됩니다.
- 비상업 소재가 하나라도 섞인 결과물은 상업 목적으로 사용할 수 없습니다.
- 유료 구독이나 소재 결제만으로 모든 내부 소재의 무제한 상업 이용권이 생기지 않습니다.
- 음악은 권리·지역·기간과 플랫폼별 조건을 개별 확인해야 합니다.

### DaVinci Resolve 21

- DaVinci Resolve 21 무료판이 제공됩니다.
- DaVinci Resolve Studio 21의 미국 공식 가격은 295달러입니다.
- 편집 프로그램 사용권과 영상에 포함한 음악·영상·폰트·플러그인의 이용권은 별개입니다.

## 추정 또는 편집 판단

다음은 공식 보장 기준이 아닙니다.

- AI를 사용하면 제작시간과 비용이 항상 절반 이하로 줄어든다는 주장
- AI 출력의 저작권이 모든 국가에서 자동으로 완전하게 성립한다는 주장
- 고객이 제공한 자료는 모두 적법하게 사용할 수 있다는 가정
- 조회수·매출·검색순위·승인을 납품 결과로 보장하는 것
- 수정 횟수와 검수시간을 제외한 콘텐츠 개수만으로 수익성을 계산하는 것

프로젝트 총시간, 수정횟수, 검수시간과 재계약률을 핵심 지표로 보는 것은 사업성 판단입니다.

## 위험과 주의사항

1. 고객의 비공개 자료를 입력하기 전 사용 제품의 데이터 처리 조건과 고객 동의를 확인합니다.
2. 개인용 계정과 비즈니스·API의 학습·보존 조건을 같은 것으로 설명하지 않습니다.
3. 고객이 제공한 자료도 권리 확인 없이 사용하지 않습니다.
4. CapCut Pro 결제와 내부 소재의 상업 이용 가능 여부를 분리합니다.
5. 조회수·매출·승인·검색순위를 보장하지 않습니다.
6. 완료본, 편집 원본, 소스파일, 프롬프트, 포트폴리오 공개 권리를 계약서에서 구분합니다.
