## 핵심 결론
이 모델의 병목은 검증된 단위이익, 파트너가 넘겨받을 수 있는 업무 절차, 독립 손익과 품질 유지, 확정수익 배분, 협업도구·교육·검수·지원비 통제입니다.

## 성패를 가르는 것
- 최근 고객의 환불·CS·작업시간을 포함한 단위이익을 먼저 검증합니다.
- 업무를 유입·상담·계약·입력·생산·검수·승인·정산·CS·재구매로 나눕니다.
- 각 단계의 담당자·입력·출력·완료 조건·예외·시간·비용을 적습니다.
- 첫 협력 형태 하나를 선택하고 고객·브랜드·콘텐츠·데이터의 사용 범위를 문서화합니다.
- 수익배분은 고객 결제액보다 환불·공통비용을 반영한 확정수익을 기준으로 검토합니다.

## 공식 확인 사실
- 네이버웍스 Free는 0원이며 최대 30명과 공용 5GB를 제공합니다.
- Lite는 월간 계약 기준 1명당 월 4,000원, Standard는 8,500원, Standard Plus는 13,000원이며 세금은 별도입니다.
- 네이버웍스는 30일 무료체험을 제공하며 Free와 체험 기간에는 SLA를 보장하지 않습니다.
- 구성원은 추가한 날부터 일할 과금되고 사용하지 않거나 첫 로그인을 하지 않아도 과금될 수 있습니다.
- 카카오워크 Free는 최대 30명까지 0원입니다.
- 카카오워크 월간 기준 Mini는 1명당 2,900원, Standard는 7,900원, Premium은 11,900원이며 부가가치세는 별도입니다.
- OpenAI의 비즈니스·API 데이터는 기본적으로 모델 학습에 사용되지 않습니다.
- OpenAI API는 ChatGPT 구독과 별도 사용량 과금입니다.
- 2026년 6월 22일 표준 처리 기준 GPT-5.5는 입력 100만 토큰당 5달러, 캐시 입력 0.50달러, 출력 30달러입니다.
- GPT-5.4 mini는 입력 0.75달러, 캐시 입력 0.075달러, 출력 4.50달러입니다.
- 웹 검색은 1,000회당 10달러이며 Batch API는 입력·출력 비용을 50% 낮춥니다.
- 공동사업과 수익배분에 모든 사업자에게 공통 적용되는 표준 비율은 공식 확인 불가입니다.
