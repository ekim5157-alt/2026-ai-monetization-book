## 핵심 결론

당근의 사업 기회는 개인 물품 처분이나 단기 일자리 중개가 아닙니다. 이 책에서 다룰 당근은 유료광고, 피드, 커뮤니티, 선생님 프로필, 비즈니스 프로필, 후기·평가·이웃·팔로우, 개인샵과 외부 링크가 한 앱 안에서 이어지는 신뢰 기반 전환 플랫폼입니다.

핵심 수익축은 세 가지입니다.

1. 당근 유료광고 기능을 직접 구매해 정수기·통신 결합상품처럼 상담과 계약이 필요한 제휴 상품을 광고하는 운영 사업
2. 프로필·개인 게시판·커뮤니티·비즈니스 프로필을 이용해 개인샵과 전국 판매로 연결하는 커머스
3. 지역 티칭으로 먼저 신뢰와 사례를 만든 뒤 외부 강의 플랫폼·전자책·상담으로 온라인화하는 교육 사업

첫 번째 축은 콘텐츠를 먼저 쌓고 프로필로 보내는 방식이 아닙니다. 운영자가 당근의 유료광고 상품을 구매하고, 인스타그램 광고 게시물처럼 이미지·문구·행동 유도 문장을 갖춘 광고 소재를 피드에 노출해 클릭이나 문의를 직접 만듭니다. 프로필과 기존 활동은 전환을 보강할 수 있지만 필수 선행 단계는 아닙니다.

유료광고의 핵심 흐름은 다음과 같습니다.

`광고비 집행 → 광고 소재 등록 → 피드 광고 노출 → 클릭·문의 → 유효 리드 → 상담 → 계약 → 설치·개통 → 승인 수익`

커머스와 교육은 다음 흐름으로 연결됩니다.

`피드·커뮤니티 콘텐츠 → 프로필 신뢰 → 개인샵 이동·수강 신청 → 구매·수강 → 후기·평가 축적 → 반복 전환`

## 성패를 가르는 것

### 1. 유료광고 운영 사업: 프로필보다 광고 소재와 승인 수익

- 구체적 관찰: 운영자가 당근의 유료광고 기능을 구매하고, 인스타그램 광고 게시물처럼 제휴 상품 광고를 피드에 직접 노출할 수 있습니다.
- 남들이 놓치는 숨은 변수: 광고 상품 유형, 집행 예산, 노출 단가, 광고 소재의 클릭 유도력, 문의자의 실제 구매 의도, 중복 리드, 상담 속도, 판매 파트너의 계약 전환 능력, 설치·개통 완료, 철회·취소, 수익 귀속 기간입니다.
- 결과가 달라지는 작동 메커니즘: 광고비로 피드 노출을 확보하고, 광고 소재가 클릭·문의를 만들며, 상담 파트너가 이를 계약과 설치·개통으로 전환해야 수익이 확정됩니다.
- 실행 판단: `광고비 → 노출 → 클릭 → 문의 → 유효 리드 → 계약 → 설치·개통 → 확정 수익`을 캠페인별로 추적합니다.
- 실패·전환 신호: 노출은 나오지만 클릭이 없으면 소재 문제입니다. 클릭은 나오지만 문의가 없으면 상품 약속이나 행동 유도가 약한 것입니다. 문의는 많지만 계약이 적으면 타깃·상담·상품 조건을 다시 봐야 합니다.

이 모델은 “게시물에 링크를 붙이는 부업”이 아니라 다음 운영 구조를 갖춘 광고 사업이어야 합니다.

- 당근 유료광고 상품 선택
- 캠페인별 예산과 중단 기준
- 상품별 광고 소재 제작
- 이미지·문구·행동 유도 문장 비교
- 클릭·문의 경로 설계
- 문의 분류와 응답 스크립트
- 상담 파트너 또는 판매 조직 연결
- 리드 중복·귀속·승인·취소 기록
- 상품별 확정 수익과 입금 시차 관리

프로필, 게시판, 커뮤니티 활동은 광고주의 신뢰를 보강할 수 있지만 광고 집행의 필수 선행 단계로 쓰지 않습니다. 광고 자체의 노출·클릭·문의·계약 성과를 별도로 계산합니다.

### 2. 당근 내 커머스: 게시물이 아니라 신뢰 그래프와 판매 동선

- 구체적 관찰: 당근 안에서는 개인 게시물, 커뮤니티, 프로필, 비즈니스 프로필, 후기와 팔로우가 서로 분리되지 않고 반복 노출됩니다.
- 남들이 놓치는 숨은 변수: 프로필 방문률, 팔로우 전환, 후기 품질, 콘텐츠와 상품의 연결성, 개인샵 이동률, 택배비, 반품·교환, 결제·정산 시차입니다.
- 결과가 달라지는 작동 메커니즘: 유용한 게시물과 커뮤니티 활동이 프로필 신뢰를 만들고, 신뢰가 개인샵 방문과 구매를 낮은 마찰로 연결합니다. 직거래에서 쌓인 평판은 택배와 전국 판매의 구매 불안을 낮출 수 있습니다.
- 실행 판단: 단일 상품 게시의 조회수보다 `콘텐츠 노출 → 프로필 방문 → 팔로우 → 개인샵 이동 → 주문 → 수령 → 반복 구매`를 측정합니다.
- 실패·전환 신호: 프로필 방문은 늘지만 샵 이동이 없으면 상품 연결이 약한 것입니다. 주문은 생기지만 배송비·반품·CS로 마진이 사라지면 상품군을 바꿔야 합니다.

이 책에서 커머스는 지역 실물 매장 홍보로 제한하지 않습니다. 당근 안에서 만든 신뢰를 개인샵, 택배 판매, 외부 결제 페이지와 연결해 전국 단위 판매로 확장하는 구조를 다룹니다.

### 3. 지역 티칭의 온라인화: 오프라인 수업이 최종 상품이 아님

- 구체적 관찰: AI 활용, 스마트폰 활용, 취미 수업, 시니어 교육은 가까운 곳에서 직접 설명하고 질문을 받는 방식이 초기 신뢰 형성에 유리합니다.
- 남들이 놓치는 숨은 변수: 반복 가능한 커리큘럼, 질문 패턴, 수강생 결과, 후기 확보, 촬영·편집 시간, 외부 강의 플랫폼 수수료, 전자책 구매 전환입니다.
- 결과가 달라지는 작동 메커니즘: 소규모 수업에서 실제 질문과 결과를 수집하고, 이를 강의 영상·워크북·전자책·상담 상품으로 재구성하면 강사의 시간을 매번 동일하게 투입하지 않아도 됩니다.
- 실행 판단: 오프라인 수업 매출보다 `수업 → 후기·사례 → 커리큘럼 표준화 → 전자책·외부 강의 → 상담·후속 상품` 전환을 봅니다.
- 실패·전환 신호: 수업은 열리지만 자료가 축적되지 않으면 계속 시간 판매에 머뭅니다. 전자책과 온라인 강의가 팔리지 않으면 현장 수요와 온라인 상품의 약속이 연결되지 않은 것입니다.

지역 티칭은 최종 수익모델이 아니라 신뢰 획득과 상품 검증의 전단계로 사용합니다. 최종 목표는 다음과 같습니다.

- 외부 강의 플랫폼의 녹화 강의
- 전자책·PDF·워크북
- 유료 상담과 소규모 코칭
- 반복 수강용 커뮤니티
- 교육 주제와 연결된 자기 상품 또는 어필리에이트

### 4. 세 축을 묶는 전환 구조

세 사업은 서로 연결될 수 있지만 유료광고는 독립적으로 성과를 계산합니다.

- 유료광고는 광고비 집행에서 시작해 클릭·문의·계약·승인 수익으로 끝납니다.
- 커머스는 게시물·커뮤니티·프로필 신뢰에서 시작해 개인샵 주문과 반복 구매로 이어집니다.
- 교육은 소규모 수업과 후기에서 시작해 전자책·외부 강의·상담으로 확장됩니다.
- 교육과 커머스에서 쌓인 후기·평가·팔로우는 유료광고 전환을 보강할 수 있지만 광고 성과와 섞어 계산하지 않습니다.

## 핵심 현실 인사이트

- 구체적 관찰: 당근 유료광고는 운영자가 광고비를 지불해 피드 노출을 사는 구조이며, 일반 게시물과 비슷한 형태의 광고 소재로 클릭과 문의를 만듭니다.
- 남들이 놓치는 숨은 변수: 프로필 신뢰보다 광고 소재, 예산, 클릭·문의 경로, 유효 리드와 승인 수익이 우선입니다.
- 결과가 달라지는 작동 메커니즘: 유료 노출이 클릭·문의를 만들고, 상담과 계약·설치·개통이 완료돼야 수익이 확정됩니다.
- 임계점·시간차·교환비용: 광고비는 즉시 지출되지만 수익은 계약·설치·취소 가능 기간을 지나 뒤늦게 확정될 수 있습니다.
- 초보자가 먼저 측정하거나 결정할 것: 일일 예산, 광고 소재 수, 클릭당 비용, 문의당 비용, 유효 리드당 비용, 승인 계약당 순이익입니다.
- 계속할 조건: 광고비 대비 승인 수익이 남고, 소재 교체로 클릭·문의·계약률이 개선되어야 합니다.
- 중단·전환 신호: 광고비는 소진되지만 유효 리드와 승인 수익이 나오지 않으면 캠페인을 중단합니다.
- 대표 사례·수치·반례: 공식 자료와 운영 사례를 조사 단계에서 확인합니다. 현재 숫자는 내부 검증 기준으로만 사용합니다.
- 반복 확인된 구조인지 단일 사례의 편집 판단인지: 유료광고 직접 집행 구조는 핵심 모델이며, 실제 광고 상품·과금·전환율은 공식 확인과 테스트가 필요합니다.

## 수익 전까지 필요한 작업량과 기간

### 유료광고·어필리에이트

`상품 1개 선정 → 광고 가능 여부 확인 → 유료광고 상품 구매 → 광고 소재 3종 제작 → 소액 집행 → 클릭·문의 수집 → 유효 리드 20건 → 계약·설치·취소 추적 → 확정 수익 계산`

초기 판단 기준:
- 집행 광고비
- 광고 노출 수
- 클릭률과 클릭당 비용
- 문의율과 문의당 비용
- 문의 대비 유효 리드 비율
- 유효 리드 대비 계약률
- 계약 대비 설치·개통 완료율
- 취소·철회율
- 승인 수익까지 걸린 기간
- 광고비와 상담비를 뺀 순이익

### 커머스

`상품군 1개 → 프로필·게시판·커뮤니티 콘텐츠 → 개인샵 연결 → 주문 10건 → 배송·반품·CS 기록 → 반복 구매 확인`

초기 판단 기준:
- 프로필 방문 대비 샵 이동률
- 샵 방문 대비 주문률
- 주문당 공헌이익
- 배송비·포장시간
- 취소·반품률
- 반복 구매율

### 티칭의 온라인화

`소규모 수업 1개 → 수강생 질문·결과 수집 → 후기 확보 → 커리큘럼 표준화 → 전자책 또는 미니 강의 → 외부 판매 테스트`

초기 판단 기준:
- 수업 신청률과 참석률
- 수료·후기 확보율
- 반복 질문 수
- 전자책·강의 제작시간
- 현장 참여자 대비 온라인 상품 구매율
- 외부 플랫폼 수수료 차감 후 순이익

## 첫 수익과 실제 입금의 시간 차이

### 유료광고·어필리에이트

`광고비 결제 → 피드 노출 → 클릭·문의 → 유효 리드 → 상담 → 계약 → 설치·개통 → 취소 가능 기간 → 수익 승인 → 지급`

광고 노출, 클릭, 문의와 계약은 아직 확정 수익이 아닙니다. 상품에 따라 설치·개통, 유지 조건, 취소·철회 기간을 지나야 수익이 확정될 수 있습니다.

### 커머스

`콘텐츠·프로필 → 개인샵 이동 → 주문·결제 → 택배 발송 → 수령 → 취소·반품 → 정산 → 입금`

직거래의 즉시 수령과 택배·개인샵 판매의 정산 구조를 섞지 않습니다. 이 책의 핵심은 반복 가능한 전국 커머스이므로 택배·반품·정산 비용을 반드시 포함합니다.

### 교육 온라인화

`지역 수업 → 후기·사례 → 전자책·강의 제작 → 외부 플랫폼 판매 → 환불 가능 기간 → 수수료 차감 → 정산·입금`

수업료 수령과 온라인 상품의 확정 수익을 분리합니다.

## 시장 현실

- 실제 구매자: 상담형 고단가 상품의 잠재고객, 신뢰할 수 있는 판매자를 찾는 온라인 구매자, 실용 교육과 디지털 자료를 찾는 학습자
- 발견·유입 경로: 유료 피드 광고, 커뮤니티, 개인 게시판, 선생님 프로필, 비즈니스 프로필, 후기·평가, 팔로우, 검색
- 판매 또는 광고가 일어나는 플랫폼: 당근 유료광고, 당근 안의 게시·프로필 기능과 정책상 허용되는 개인샵·외부 강의·전자책·결제 경로
- 결제수단: 광고비 결제수단과 각 판매처·제휴사가 지원하는 카드, 간편결제, 계좌이체 등
- 수익 지급자: 어필리에이트·광고 제휴사, 상품 구매자, 개인샵 또는 결제대행사, 외부 강의·전자책 플랫폼
- 비용: 광고비, 광고 소재 제작비, 상담 인건비, 리드 중복·부정 비용, 상품 원가, 포장·배송, 반품·CS, 강의 제작비, 플랫폼 수수료, 세금
- 반복 조건: 유료광고는 승인 수익이 광고비보다 커야 하고, 커머스와 교육은 반복 구매·수강과 비용 차감 순이익이 남아야 합니다.

## 판매·수익·광고 경로

### 유료광고 운영 사업

`당근 유료광고 구매 → 광고 소재 등록 → 피드 노출 → 클릭·문의 → 상담 파트너 → 계약 → 설치·개통 → 승인 → 제휴 수익`

정수기·통신 결합상품 같은 예시는 대표적인 상담형 상품입니다. 공식 정책과 광고·제휴 계약에서 허용되는 상품만 다룹니다.

### 당근 내 커머스

`게시물·커뮤니티 → 프로필·팔로우 → 개인샵 → 주문·결제 → 택배·수령 → 반품 기간 → 정산`

당근 안에서 신뢰를 만들고 판매는 개인샵 또는 허용된 결제 경로에서 전국 단위로 완결합니다.

### 지역 티칭의 온라인화

`지역 수업·모임 → 후기·사례 → 선생님 프로필 → 전자책·워크북·외부 강의 → 상담·후속 상품`

오프라인 수업 횟수를 늘리는 대신, 현장 질문과 결과를 재사용 가능한 디지털 상품으로 바꾸는 것이 핵심입니다.

## 공식 확인 사실

사업성 분석 단계에서는 다음을 확정하지 않습니다.

- 당근 유료광고 상품의 유형·과금·노출·타기팅·심사·환불 구조
- 어필리에이트 상품과 유료광고의 허용 범위
- 광고에서 링크·전화·문의로 전환하는 기능
- 상담형 상품의 리드 귀속·계약 승인·취소·지급 조건
- 비즈니스 프로필과 개인샵의 실제 기능·노출·결제·정산 구조
- 외부 강의·전자책 플랫폼 연결 허용 범위
- 광고 표시, 반복 게시, 삭제·제재 기준

다음 조사 단계가 당근 공식 광고 안내·도움말·운영정책과 각 제휴사·판매처·강의 플랫폼의 공식 문서로 확인해야 합니다.

## 현실 기준 교정

- 당근 유료광고는 콘텐츠를 먼저 쌓아 프로필로 보내는 모델이 아닙니다.
- 운영자가 광고비를 지불해 광고 소재를 피드에 노출하고 클릭·문의·계약 전환을 직접 노리는 구조입니다.
- 프로필·커뮤니티 신뢰는 유료광고 성과를 보강할 수 있지만 필수 선행 단계가 아닙니다.
- 유료광고는 클릭 수익이 아니라 광고비·유효 리드·계약·설치·승인·취소를 포함한 운영 사업으로 봅니다.
- 커머스는 프로필·게시판·커뮤니티·개인샵을 통한 전국 판매로 봅니다.
- 티칭은 오프라인 시간 판매가 아니라 후기·사례·커리큘럼을 외부 강의·전자책으로 전환하는 과정으로 봅니다.
- 세 축은 서로 연결될 수 있지만 각 수익의 확정 조건과 입금 시차는 별도로 계산합니다.

## 추정 또는 편집 판단

- 당근 유료광고를 인스타그램 광고 게시물과 유사한 피드형 광고 운영으로 보는 것은 사용자 관찰을 기반으로 한 편집 관점입니다.
- 정수기·통신 결합상품을 대표 사례로 드는 것은 상담형 고단가 어필리에이트 구조를 설명하기 위한 예시입니다.
- 실제 광고 상품·링크 기능·노출 방식·전환율은 공식 조사와 테스트가 필요합니다.
- 전국 커머스와 교육 온라인화의 실제 전환율은 운영 사례와 테스트가 필요합니다.
