## 핵심 결론

당근은 중고거래 앱에 머물지 않습니다. 지역 생활권 안에서 개인·소상공인·프리랜서가 고객을 발견하고, 가까운 거리와 반복 노출을 바탕으로 신뢰를 쌓아 채팅·방문·예약·거래로 전환하는 생활 플랫폼입니다.

`생활권 수요 × 거리·시간 편의 × 프로필 신뢰 × 채팅 응답 × 거래 완료·재거래 × 광고비·이동비·응대시간 차감 후 순이익`

당근에서는 조회보다 채팅 시작률, 약속 확정률, 거래 완료율, 재거래율, 노쇼율, 이동시간, 고객당 순이익이 중요합니다.

## 성패를 가르는 것

### 생활권 안의 급한 문제
- 관찰: 가까운 곳에서 지금 해결할 수 있는 물건·서비스·정보 수요가 강합니다.
- 숨은 변수: 이동 가능 거리, 제공 시간, 방문·픽업·출장 방식, 문제의 긴급성, 반복 거래 가능성입니다.
- 작동 메커니즘: 거리가 가까울수록 방문·픽업·출장 마찰이 줄어듭니다.
- 실행 판단: `어느 생활권의 누가, 언제, 어떤 문제를, 얼마나 가까운 거리에서 해결받는가`를 한 문장으로 정합니다.
- 실패·전환 신호: 지역·시간 불일치와 이동비가 반복되면 범위·시간·최소 단가를 조정합니다.

### 프로필 신뢰
- 관찰: 친근함만으로 서비스·상점 거래가 성립하지는 않습니다.
- 숨은 변수: 경력, 가격, 작업 범위, 실제 사례, 가능한 지역·시간, 취소·환불 기준입니다.
- 작동 메커니즘: 누가 무엇을 얼마에 어떻게 제공하는지 보이면 불필요한 문의가 줄어듭니다.
- 실행 판단: 대상, 문제, 기본 가격, 지역·시간, 포함·제외 범위, 실제 사례를 명시합니다.
- 실패·전환 신호: 가격·범위 질문만 반복되면 비즈프로필과 게시물 구조를 다시 설계합니다.

### 채팅에서 거래 완료까지
- 관찰: “관심 있어요”와 가격 문의는 매출이 아닙니다.
- 숨은 변수: 첫 답변시간, 반복 질문, 일정 확정, 노쇼, 당일 취소, 거래 완료 확인입니다.
- 작동 메커니즘: 가격·시간·장소·준비물이 정리되면 채팅이 약속으로 전환됩니다.
- 실행 판단: `문의 확인 → 지역·일정 → 가격·범위 → 약속 확정 → 거래 완료 기록`으로 고정합니다.
- 실패·전환 신호: 채팅은 많지만 약속·거래가 적으면 답변 구조와 예약·취소 기준을 바꿉니다.

### 지역광고
- 관찰: 노출과 클릭이 늘어도 생활권·시간·가격이 맞지 않으면 거래가 생기지 않습니다.
- 숨은 변수: 광고 지역 크기, 실제 이동 반경, 소재, 채팅 시작률, 거래당 광고비, 재거래율입니다.
- 작동 메커니즘: 지역을 넓히면 관심은 늘지만 이동비·취소도 늘 수 있습니다.
- 실행 판단: `노출 → 프로필 방문 → 채팅 → 약속 → 거래 → 재거래`를 생활권별로 기록합니다.
- 실패·전환 신호: 클릭비용은 낮지만 거래 완료와 재거래가 없으면 광고 지역·소재·가격을 바꿉니다.

### 거래 유형 분리
- 관찰: 개인 물품 처분, 반복 상품 판매, 서비스 의뢰는 책임·정산·세금 구조가 다릅니다.
- 숨은 변수: 일회성 여부, 재고·배송·반품, 사업자 조건, 서비스 책임, 예약정보, 증빙 발급입니다.
- 작동 메커니즘: 중고거래는 물품 상태와 직거래 안전, 상품 판매는 정보·환불·세금, 서비스는 범위·일정·취소가 핵심입니다.
- 실행 판단: 개인 물품, 상품·재고, 방문·출장·레슨·제작 서비스를 별도 장부로 관리합니다.
- 실패·전환 신호: 개인 거래와 사업 홍보가 섞이거나 거래 유형별 순이익을 구분하지 못하면 구조를 분리합니다.

### 플랫폼 밖 전환
- 관찰: 발견·채팅 뒤 결제·예약·장기 고객관리에 다른 도구가 필요할 수 있습니다.
- 숨은 변수: 외부 링크, 전화번호·이메일, 예약·결제 연결, 반복 홍보, 개인정보 보관입니다.
- 작동 메커니즘: 외부 단계가 길면 이탈하고 정책을 위반하면 노출·계정이 제한될 수 있습니다.
- 실행 판단: 당근 안에서 충분히 설명하고 꼭 필요한 경우에만 공식 정책이 허용하는 방식으로 연결합니다.
- 실패·전환 신호: 외부 결제 요구가 신뢰를 떨어뜨리거나 게시물 삭제·경고가 반복되면 중단합니다.

## 핵심 현실 인사이트
- 대규모 이용자 기반보다 생활권 안의 거리·시간·신뢰가 거래를 좌우합니다.
- 이동시간, 노쇼, 채팅 응대시간, 거래 유형, 생활권별 고객획득비용이 숨은 변수입니다.
- 가까운 문제를 명확한 가격·범위·시간으로 제시하면 채팅이 약속과 거래로 이어집니다.
- `게시물·광고 → 프로필 → 채팅 → 약속 → 거래·서비스 → 취소·환불 → 비용·세금 → 원화 계좌 입금`
- 계속할 조건은 거래 완료율과 재거래가 올라가고 비용 차감 순이익이 남는 것입니다.
- 중단 신호는 채팅만 늘고 거래가 늘지 않거나 노쇼·이동비가 마진을 없애는 경우입니다.

- 서비스 의뢰: `게시물·광고 → 채팅 → 일정·범위 합의 → 서비스 수행 → 취소·수정·환불 → 원화 계좌 입금`

## 시장 현실
- 이용자·구매자: 가까운 거리에서 물건·서비스·수업·수리·제작·상점 정보를 찾는 개인과 소상공인
- 유입: 당근 홈·검색·동네생활·비즈프로필·지역광고·채팅
- 판매 유형: 개인 중고거래, 상품·재고, 방문·출장 서비스, 클래스·상담, 콘텐츠 제작·업무 세팅
- 결제: 플랫폼이 지원하는 방식과 공식적으로 제공 가능한 신용카드·간편결제·계좌이체
- 지급자: 구매자, 최종 판매처 또는 PG사
- 비용: 광고·이동·재료·포장·배송·노쇼·채팅·플랫폼·PG수수료·세금
- 반복 조건: 같은 생활권에서 재거래·소개가 생기고 순이익이 남아야 합니다.

## 판매·수익·광고 경로
- 중고거래: 게시물 → 채팅 → 직거래·배송 → 거래 완료
- 소상공인 상품: 비즈프로필·상품·지역광고 → 채팅·방문 → 결제·수령
- 지역 서비스: 게시물·비즈프로필·지역광고 → 채팅 → 상담·견적 → 방문·출장·수업 → 결제
- 콘텐츠·자동화·세팅: 사례 게시물 → 채팅 → 유료 진단 → 계약·착수금 → 납품·잔금
- 지역광고는 생활권별 거래당 광고비와 고객당 순이익이 확인될 때만 확대합니다.

## 공식 확인 사실

사업성 분석 단계에서는 이용자 규모, 개인 계정·비즈프로필 게시 기준, 지역광고 과금·심사·환불, 외부 링크·결제·예약 허용 범위, 안심결제·송금·정산·분쟁 조건을 확정하지 않습니다. 다음 단계가 당근 공식 도움말·광고 안내·약관·결제 안내로 확인해야 합니다.

- 카카오는 이 모델에 포함하지 않습니다. 전화번호 기반 지인 연락망과 기업·기관 인프라의 성격이 강해 신규 고객 발견 플랫폼과 역할이 다릅니다.
- 개인 거래, 반복 상품 판매, 서비스 의뢰를 서로 다른 구조로 구분합니다.
- 채팅 수보다 약속·거래 완료·재거래·순이익을 기록합니다.
- 외부 링크·결제·예약은 공식 정책이 확인된 범위에서만 사용합니다.
- 개인과 개인사업자의 판매·광고·세금·환불 조건을 구분합니다.
