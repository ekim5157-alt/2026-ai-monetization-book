# HANDOFF

- 담당 역할: 사업성 분석
- 작성일: 2026-06-21
- 공식 확인일: 조사 담당 재검증 필요

## 핵심 결론

당근은 중고거래 앱에 머물지 않습니다. 이 책에서 다룰 당근은 **유료광고, 피드, 커뮤니티, 선생님 프로필, 비즈니스 프로필, 후기·평가·이웃·팔로우, 개인샵과 허용된 외부 링크가 결합되는 신뢰 기반 전환 플랫폼**입니다.

모델 19는 세 사업축을 연결합니다.

1. 유료광고 → 문의 → 유효 리드 → 계약 → 설치·개통 → 승인·환수
2. 콘텐츠·프로필 신뢰 → 개인샵 → 주문·택배 → 반품·정산·반복 구매
3. 커뮤니티 코칭 → 질문·성과 축적 → 강의·전자책·멤버십·상담·SaaS

`광고 순이익 + 커머스 순이익 + 교육·구독 순이익 = 당근 기반 전체 순이익`

세 축은 후기·팔로우·프로필 신뢰를 공유하지만 수익과 비용 장부는 분리합니다.

## 성패를 가르는 것

### 1. 세 사업축의 신뢰는 연결하고 수익 귀속은 분리합니다
- 관찰: 피드·커뮤니티·프로필·후기는 광고·커머스·교육 모두의 전환을 보강할 수 있습니다.
- 숨은 변수: 문의가 광고·개인샵·커뮤니티 중 어디서 시작됐는지, 후기·팔로우가 어느 전환에 기여했는지입니다.
- 작동 메커니즘: 공통 신뢰 자산은 전환 마찰을 낮추지만 광고비를 쓴 리드와 자연 주문을 섞으면 순이익을 오판합니다.
- 실행 판단: 모든 문의·주문·신청에 광고, 피드·커뮤니티, 비즈니스 프로필, 개인샵, 선생님 프로필, 외부 콘텐츠, 재구매 태그를 붙입니다.
- 실패·전환 신호: 문의는 늘지만 수익 경로와 비용을 구분하지 못하면 사업축별 장부부터 다시 만듭니다.

### 2. 유료광고는 링크 부업이 아니라 캠페인 운영 사업입니다
- 관찰: 운영자가 광고 상품을 직접 구매하고 이미지·문구·행동 유도 문장을 갖춘 소재로 문의와 상담을 만들 수 있습니다.
- 숨은 변수: 광고 상품·과금·예산·타기팅·심사, 클릭당 비용, 문의당 비용, 무효 리드, 상담 계약률, 설치·개통, 취소·철회·환수입니다.
- 작동 메커니즘: 클릭이 문의가 되고 유효 리드·계약·설치·개통·승인 단계를 통과해야 수익이 확정됩니다.
- 실행 판단: `광고비 → 노출 → 클릭 → 문의 → 유효 리드 → 계약 → 설치·개통 → 승인 → 환수 반영 순이익`을 캠페인별로 기록합니다.
- 실패·전환 신호: 문의는 많지만 무효 리드가 높거나 승인 수익이 광고비·소재비·상담비보다 작으면 예산을 늘리지 않습니다.

대표 후보는 정수기, 인터넷·통신 결합상품, 상담과 설치가 필요한 생활 서비스입니다.

### 3. 광고 소재와 프로필 신뢰를 별도 측정합니다
- 관찰: 프로필·후기·커뮤니티 활동은 신뢰를 높이지만 광고 성과를 대신하지 않습니다.
- 숨은 변수: 첫 이미지, 대상·조건 문구, 상담 신청 마찰, 후기와 광고 약속의 일치, 리드 개인정보 동의입니다.
- 작동 메커니즘: 대상·조건·진행 과정·제외 조건이 구체적이면 낮은 의도의 문의가 줄어듭니다.
- 실행 판단: 광고 소재는 대상, 문제, 핵심 조건, 신청 뒤 과정, 취소·제외 조건으로 구성합니다.
- 실패·전환 신호: 혜택만 강조해 낮은 의도 문의가 몰리거나 광고 약속과 상담 조건이 다르면 소재를 바꿉니다.

### 4. 당근 내 커머스는 지역 매장 홍보에 제한되지 않습니다
- 관찰: 직거래에서 쌓인 후기·평가·이웃·팔로우·거래 이력은 개인샵과 택배 판매에서도 구매 불안을 줄일 수 있습니다.
- 숨은 변수: 프로필 방문, 팔로우, 개인샵 이동, 주문 전환, 원가·판매가, 포장·택배비, 결제수수료, 반품·분쟁, 반복 구매입니다.
- 작동 메커니즘: 피드·커뮤니티 신뢰가 개인샵 주문을 만들고 배송 경험이 다시 후기와 팔로우를 늘리면 전국 판매로 확장될 수 있습니다.
- 실행 판단: 직거래는 이동·응대시간과 실제 수령액, 택배는 포장·배송·반품·CS·정산, 개인샵은 방문·주문·반복 구매를 별도 계산합니다.
- 실패·전환 신호: 팔로우는 늘지만 개인샵 이동이 없거나 택배·반품 비용이 마진을 없애면 상품·가격·판매 범위를 바꿉니다.

`콘텐츠·커뮤니티 → 프로필 신뢰 → 개인샵 이동 → 주문·결제 → 택배·수령 → 반품 기간 → 정산`

### 5. 커뮤니티 코칭은 교육 플랫폼으로 확장해야 합니다
- 관찰: 동네 모임과 직접 티칭은 신뢰와 질문·실패 패턴·성과 사례를 모으는 진입 방식입니다.
- 숨은 변수: 모집 대비 참여, 재참여, 과제 제출, 결과물 완성, 반복 질문, 후기, 커리큘럼 재사용, 피드백시간, 후속 상품 전환입니다.
- 작동 메커니즘: 질문·과제·결과물·실패 패턴을 표준화하면 직접 수업이 강의·전자책·멤버십·상담·SaaS로 바뀝니다.
- 실행 판단: 주간 목표·과제·자료·피드백·진행 확인·결과 발표·후기 동의·다음 상품 제안을 운영 구조로 고정합니다.
- 실패·전환 신호: 참여·과제·결과가 낮거나 피드백시간이 참가비를 넘고 후속 상품 전환이 없으면 주제와 운영 방식을 바꿉니다.

`커뮤니티 모집 → 코칭·실습 → 질문·성과·후기 → 커리큘럼·도구 표준화 → 강의·전자책·멤버십·상담·SaaS`

### 6. SaaS는 커뮤니티에서 반복 확인된 문제에서 나와야 합니다
- 관찰: 커뮤니티 뒤에 SaaS를 붙이는 것 자체는 확장이 아닙니다.
- 숨은 변수: 반복 질문 빈도, 기존 해결시간, 체험 뒤 재사용, 사용자당 비용, 유료 전환·해지, 자료 사용 동의입니다.
- 작동 메커니즘: 커뮤니티가 제품 리서치와 초기 사용자 집단이 되면 문제 선택 실패를 줄일 수 있습니다.
- 실행 판단: 반복 질문을 빈도·시간·오류비용으로 순위화하고 한 문제만 간단한 도구로 시험합니다.
- 실패·전환 신호: 주간 재사용이 없거나 사용자당 비용이 결제액보다 크면 개발을 확대하지 않습니다.

### 7. 원소스멀티유즈는 세 사업축의 수요를 연결합니다
- 관찰: 광고 소재, 상품 시연, 커뮤니티 과제·성과는 하나의 원본에서 채널별 콘텐츠로 만들 수 있습니다.
- 숨은 변수: 경제적 이해관계 표시, 사례 동의, 채널별 첫 장면·길이·링크 목적, 당근으로 돌아오는 경로, 제작시간입니다.
- 작동 메커니즘: 같은 원본을 유튜브·인스타그램·틱톡·X·페이스북에 맞춰 배포하면 당근 밖에서도 신뢰와 수요를 만들 수 있습니다.
- 실행 판단: 유튜브는 긴 설명, 인스타그램은 시각 사례, 틱톡은 빠른 과정, X는 핵심 수치·관점, 페이스북은 지역·관심사 그룹 사례로 재가공합니다.
- 실패·전환 신호: 채널 수만 늘고 프로필·개인샵·문의·커뮤니티 신청이 늘지 않거나 제작비가 추가 수익을 넘으면 채널을 줄입니다.

## 핵심 현실 인사이트
- 구체적 관찰: 핵심은 지역성 자체보다 직거래·참여에서 만든 신뢰가 광고 상담, 개인샵 주문, 교육·구독 전환으로 이동하는가입니다.
- 남들이 놓치는 숨은 변수: 광고 승인·환수, 개인샵 이동·반복 구매, 과제·후속 상품 전환, 후기·팔로우의 수익축별 영향입니다.
- 결과가 달라지는 작동 메커니즘: 프로필·후기·거래·참여 이력이 세 축의 전환 마찰을 낮춥니다.
- 임계점·시간차·교환비용: 광고는 설치·승인, 커머스는 배송·반품, 교육은 참여·결과·후속 상품 뒤 수익이 확정됩니다.
- 초보자가 먼저 측정하거나 결정할 것: 첫 사업축 1개, 유입 태그, 비용 장부, 최종 수익 조건입니다.
- 계속할 조건: 한 축의 순이익이 확인되고 공통 신뢰 자산이 다른 축의 전환도 보강하는 경우입니다.
- 중단·전환 신호: 노출·문의·팔로우는 늘지만 승인 수익·구매확정·유료 전환이 늘지 않는 경우입니다.
- 대표 사례·수치·반례: 광고 클릭 수천 회보다 설치·개통 완료 계약이 중요하고 커뮤니티 참가자 수보다 결과물과 후속 구매가 중요합니다.
- 반복 확인된 구조인지 단일 사례의 편집 판단인지: 세 사업축 연결은 사업성 판단이며 광고·개인샵·결제·외부 링크 조건은 공식 확인이 필요합니다.

## 시장 현실
- 네이버 블로그·카페·브런치·유튜브·커뮤니티에서 먼저 확인한 동향: 비즈니스 프로필·피드 광고·커뮤니티·교육·개인샵·택배 판매로 역할이 넓어지는 흐름이 보입니다.
- 반복된 패턴: 지역광고를 클릭 광고로만 보거나 커뮤니티를 무료 모임으로만 운영하거나 개인샵을 지역 매장 홍보로만 제한하는 해석이 많습니다.
- 상충되는 의견: 지역성 집중과 후기·팔로우 신뢰의 전국 커머스 확장 가능성이 공존하므로 직거래·택배·주문 전환을 확인해야 합니다.
- 작성자 이해관계 또는 광고 가능성: 광고 운영자·강사·개인샵 판매자는 특정 기능 성과를 과장할 유인이 있습니다.
- 이미 포화된 부분: 근거 없는 생활정보 반복, 가격만 강조한 게시물, 무료 모임 뒤 무리한 판매입니다.
- 아직 남은 기회: 상담형 제휴 캠페인, 후기 기반 개인샵, 커뮤니티 질문을 교육·전자책·SaaS로 바꾸는 연결 구조입니다.

## 판매·수익·광고 경로
- 실제 구매자·독자·시청자·광고주: 생활 서비스 상담 수요, 상품 구매자, 스터디·취미·실습 참가자, 소상공인·프리랜서·강사
- 발견·유입 경로: 당근 피드·커뮤니티·검색·유료광고·선생님 프로필·비즈니스 프로필·후기·평가·이웃·팔로우, 다섯 영상 채널
- 판매 또는 광고가 일어나는 플랫폼: 당근 유료광고, 개인샵·비즈니스 프로필·허용된 판매 경로, 클래스101·탈잉·크몽, 브런치스토리·교보문고·YES24·알라딘, 자체 판매페이지
- 결제수단: 당근과 연결 판매처가 지원하는 결제, 신용카드·간편결제·계좌이체
- 수익 지급자: 상담형 제휴 운영사, 상품 판매 플랫폼·PG사, 강의·전자책·구독 판매처
- 플랫폼 수수료·광고비·환불·CS 비용: 광고비·소재비·상담비·환수, 상품 원가·포장·배송·반품·CS, 커뮤니티 운영·강의 제작·플랫폼 수수료·SaaS 비용
- 확정 수익 조건: 광고는 계약·설치·개통·승인, 커머스는 배송·반품·구매확정, 교육은 환불·수강·구독 조건을 통과한 경우
- 실제 입금 시점과 경로: 제휴사·판매처·강의 플랫폼·PG 정산을 거쳐 원화 계좌 입금
- 반복 가능한 구조인지: 캠페인 순이익, 개인샵 반복 구매, 커뮤니티 후속 전환이 각각 반복되어야 합니다.
- 대상 범위 교정: 한글 콘텐츠·원화 결제·원화 정산과 당근의 실제 허용 기능을 기준으로 합니다.

## 공식 확인 사실
