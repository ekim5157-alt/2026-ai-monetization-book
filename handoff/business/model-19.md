## 핵심 결론
당근은 중고거래만의 플랫폼이 아니다. 현재 당근비즈니스는 비즈프로필, 디스플레이 광고, 검색 광고, 네이티브 광고, 카탈로그 상품 광고, 키워드 광고, 견적, 예약, 앱 안 상품 결제를 함께 제공한다. 이 모델은 세 축으로 나눠 봐야 한다.

1. 운영자가 광고비를 직접 집행해 정수기·통신·인테리어·수리·교육 등 상담형 상품의 문의를 만드는 유료광고 운영.
2. 온라인 도매상과 연결해 농산물 박스를 판매하고, 도매상이 재고·포장·배송을 담당하는 무재고 커머스.
3. 동네 모임·커뮤니티에서 코칭을 운영한 뒤 강의·전자책·상담·멤버십·SaaS로 확장하는 교육 사업.

세 축의 돈이 확정되는 시점은 다르다. 광고의 클릭·문의는 수익이 아니고 설치·개통·철회기간 통과 뒤 승인 수익이 된다. 농산물 주문은 배송·품질·환불을 지나 판매대금이 확정된다. 커뮤니티 참가자 수는 과제 완료·재참여·유료 상품 구매로 이어져야 매출이 된다.

## 성패를 가르는 것
- 구체적 관찰: 당근 공식 페이지는 2026년 1월 기준 월간 이용자 2,000만+라는 자체 수치와 동네 기반 광고, 상품 즉시 결제, 비즈프로필을 함께 안내한다.
- 남들이 놓치는 숨은 변수: 지역별 고객 밀도, 문의 응답속도, 실제 서비스 가능 반경, 광고 뒤 상담·설치 취소, 농산물 품질 편차, 후기 누적, 커뮤니티 과제 완성률.
- 작동 메커니즘: 동네·관심사·검색어로 발견된 사람이 채팅·전화·상품 결제로 이동하고, 후기·단골·팔로우가 다음 구매의 불확실성을 낮춘다.
- 임계점·시간차·교환비용: 광고는 문의가 쌓여도 설치·개통 전에는 확정 수익이 아니다. 신선식품은 배송 후 하자·변질·반품이 끝나야 이익이 보인다. 커뮤니티는 무료 모임이 유료 상품으로 넘어가는 기준이 필요하다.
- 초보자가 먼저 측정할 것: 광고비, 노출, 클릭, 문의, 유효 문의, 상담, 계약, 설치·개통, 철회, 승인 수익. 상품은 주문·발송·배송완료·클레임·구매확정·반복 구매. 모임은 신청·출석·과제 제출·완성·재참여·유료 구매.
- 사업을 계속할 조건: 광고비와 CS·환불을 뺀 승인 수익이 남고, 상품 후기가 반복 구매를 만들며, 모임 질문이 표준 강의·도구로 축적돼야 한다.
- 중단·전환 신호: 클릭은 많은데 유효 문의가 적음, 상담 후 설치 취소가 반복됨, 공급업체 출고가 늦고 품질 편차가 큼, 리뷰가 배송·품질 불만에 집중됨, 모임 운영시간만 늘고 유료 전환이 없음.

## 시장 현실

### 축 1. 유료광고 직접 운영
| 요소 | 실제 운영 구조 |
|---|---|
| 실제 돈을 내는 사람 | 정수기·통신·인테리어·수리·교육·이사·청소 등의 최종 고객. 광고비는 운영자가 선지출 |
| 발견 경로 | 당근 홈·다양한 화면의 디스플레이 광고, 검색 결과의 검색 광고, 전문가모드의 네이티브·카탈로그·키워드 광고 |
| 행동 경로 | 광고 노출 → 클릭 → 비즈프로필·채팅·전화 → 상담 → 계약 → 설치·개통·서비스 완료 |
| 수익 지급자 | 제휴 본사·대리점·서비스 고객 또는 직접 계약 고객 |
| 확정 조건 | 유효 리드, 계약, 설치·개통, 취소·철회기간, 본사 승인 조건 통과 |
| 주요 비용 | 광고비, 이미지·문구 제작, 상담시간, 이동, 취소, 설치 실패, 콜백, 민원·환불 |

- 시장 반복 패턴: `무료 콘텐츠를 먼저 오래 쌓아야만 광고할 수 있다`는 구조가 아니다. 광고 소재와 행동 유도 문장을 바로 시험할 수 있다.
- 시장 반복 패턴: 지역 타깃은 전국 노출보다 실제 서비스 가능한 동네와 상담 인력에 맞춰야 한다.
- 초보자가 잘못 읽기 쉬운 숫자: 클릭당 비용이 낮아도 유효 문의·계약·개통당 비용이 높으면 손실이다.
- 남은 기회: 설치·방문·견적이 필요한 생활 서비스와 지역 교육. 지역별 후기와 응답속도가 큰 차별점이 된다.
- 포화된 부분: 똑같은 가격 문구·무료 상담·과도한 혜택을 반복하는 광고. 문의 뒤 응답이 늦으면 광고비가 그대로 사라진다.

### 축 2. 온라인 도매상과 연결된 농산물 박스 판매
`온라인 도매상 계약 → 품목·산지·등급·중량·포장·출고 조건 수급 → 당근 상품·게시물 등록 → 주문·결제 → 도매상 재고 확인·포장·배송 → 고객 수령 → 하자·환불 처리 → 판매대금 지급 → 후기·반복 구매`

| 주체 | 역할 |
|---|---|
| 온라인 도매상·공급사 | 농산물 확보, 등급·중량 정보, 재고, 포장, 송장, 출고, 품질·파손 협조 |
| 판매자 | 상품 선택, 산지·생산자 이야기, 가격·사진·상품 설명, 당근 운영, 문의, 후기, 재구매 |
| 당근 | 상품 노출·비즈프로필·채팅·앱 내 결제·운영정책 제공 |
| 택배사 | 집하·배송·파손·지연 정보 제공 |
| 구매자 | 상품대금 결제, 수령 후 하자 신고·후기·반복 구매 |

- 현재 공급망 확인: 온채널과 도매매는 농산물·수산물·축산물·특산품 카테고리와 위탁판매·배송대행 구조를 운영한다. 공급처가 있다는 사실과 특정 상품의 수익성이 있다는 사실은 다르다.
- 상품 욕구: 싼 가격만이 아니라 산지, 제철, 생산자, 선물, 가족용 박스, 선별 기준, 빠른 출고에 대한 신뢰.
- 발견 경로: 동네 게시물·상품·비즈프로필·후기·공유. 전국 판매 가능 범위와 배송 정책은 계정 화면 확인이 필요하다.
- 수익 확정: 주문금액에서 공급가·택배비·포장비·결제·플랫폼 비용·환불·재배송·폐기 비용을 뺀 뒤 남는 금액.
- 숨은 변수: 같은 품목도 크기·당도·숙도·상처·중량 오차가 생긴다. 사진과 실제 박스의 차이가 후기와 재구매를 무너뜨린다.
- 반복 구매: 첫 구매의 마진보다 다음 제철 품목을 다시 사는 고객 비율이 중요하다. 구매자 연락처·마케팅 사용은 개인정보·플랫폼 정책을 확인한다.
- 세금: 미가공 농산물의 면세 가능성과 가공·혼합·선물포장 상품의 과세 여부를 분리한다. 공급사 증빙과 판매자 매출 증빙을 확인한다.
- 중단 신호: 배송 지연, 품질 편차, 공급가 급변, CS 책임 떠넘김, 공급사 송장·재고 갱신 지연, 반복 구매 없음.

### 축 3. 커뮤니티 코칭의 강의·전자책·SaaS 확장
`동네 모임 모집 → 목표·기간·과제 제시 → 현장·온라인 피드백 → 질문·결과물·후기 축적 → 반복 문제를 커리큘럼·워크북·도구로 표준화 → 강의·전자책·상담·멤버십·SaaS 판매`

- 실제 구매자: 스마트폰 활용, 그림, 글쓰기, AI 활용, 운동, 취미, 자격·취업, 소상공인 실무 등 결과를 원하는 지역 주민.
- 발견 경로: 당근 모임·게시물·비즈프로필·광고·후기·지인 추천.
- 돈이 생기는 순서: 모임 참가비 → 1:1 상담·소규모 코칭 → 전자책·워크북 → 반복 강의·멤버십 → 반복 문제를 해결하는 SaaS.
- 시장 반복 패턴: 모임 인원보다 참가자의 완성 결과와 후기가 다음 모집의 핵심 자산이 된다.
- 숨은 변수: 운영자가 매회 같은 설명을 반복하면 교육 매출은 늘어도 시간이 고정된다. 질문을 문서·동영상·템플릿·도구로 바꿔야 확장된다.
- 중단 신호: 출석은 높지만 과제 제출이 없음, 결과물이 없음, 재참여가 없음, 운영자가 모든 질문을 직접 처리함, SaaS를 만들었지만 모임에서 반복된 문제가 아님.

## 공헌이익 계산
### 광고형
`승인 수익 - 광고비 - 상담·이동 시간원가 - 취소·철회 손실 - 소재 제작비 - 세금 = 순이익`

### 농산물 박스
`판매가 - 공급가 - 포장·택배비 - 플랫폼·결제 비용 - 환불·재배송·폐기 - CS 시간 - 세금 = 박스당 공헌이익`

### 커뮤니티
`참가비·상품 매출 - 장소·광고·결제비 - 교재·도구비 - 환불 - 운영·피드백 시간원가 - 세금 = 회차당 순이익`

## 초보자가 잘못 읽기 쉬운 숫자
- 노출·클릭: 문의나 계약이 아니다.
- 문의: 실제 서비스 가능 지역·예산·일정이 맞는 유효 리드와 다르다.
- 주문: 배송완료·하자·환불을 지나지 않은 매출이다.
- 후기 수: 최근 품질과 반복 구매를 자동 보장하지 않는다.
- 모임 가입자: 출석·과제·결과물·유료 구매와 다르다.
- 당근 성공사례의 매출 비중: 개별 사례이며 평균 성과로 사용하지 않는다.

## 공식 확인 사실
- 당근비즈니스는 2026년 1월 기준 자체 발표로 서비스 가입자 4,300만, 월간 이용자 2,000만+를 제시한다.
- 현재 광고 상품은 디스플레이·검색 광고와 전문가모드의 네이티브·카탈로그·키워드 광고로 나뉜다.
- 비즈프로필은 무료 업체 등록을 안내하며 견적, 예약, 상품 판매, 포장주문 도구를 연결한다.
- 당근 앱 안에서 상품을 즉시 결제할 수 있다고 안내한다. 수수료·정산·환불·판매 가능 품목은 상품판매 운영정책과 실제 계정 화면에서 확인해야 한다.
- 도매매·온채널 현재 화면에는 농산물과 지역 특산품 카테고리, 공급사·판매사 등록, 상품 DB·엑셀·배송대행 기능이 노출된다.

## 조사 결과 때문에 수정한 기존 가정
- 당근을 중고거래·개인 판매 중심으로 한정하지 않는다.
- 농산물 박스 판매를 짧은 부수 사례로 두지 않고 공급·품질·정산·재구매가 독립적으로 작동하는 축으로 둔다.
- 커뮤니티를 모임 참가비 사업으로 끝내지 않고 반복 질문을 강의·전자책·SaaS로 바꾸는 경로를 분석한다.
- 당근의 이용자 수와 성공사례를 판매자 평균 매출로 해석하지 않는다.
