# Draft handoff

전체 작업의 세 번째 단계입니다.

초보자 실행 원고를 쓰기 전에 반드시 같은 모델의 다음 파일을 읽습니다.

- `handoff/business/model-XX.md`
- `handoff/research/model-XX.md`

사업성 분석의 숨은 병목과 공식자료 검증 결과를 통합하되 그대로 복사하지 않습니다.

원고 작성 원칙:
- 기본 독자는 한국의 소비자·독자·시청자에게 판매하거나 광고·제휴·대행 수익을 만들려는 한국 거주 초보자입니다.
- 한국에서 실제 확인된 구매자, 유입 채널, 판매 장소, 결제수단, 수익 지급자, 정산 시점부터 설명합니다.
- 조회수·클릭·문의가 아니라 실제 결제와 확정 입금으로 이어지는 경로를 씁니다.
- 국내에서 작동하는 판매·수익·광고 경로가 확인되지 않은 모델은 실행 가능한 국내 모델처럼 서술하지 않습니다.
- 미국 사업자나 콘텐츠 사업자의 사례는 한국에 대응되는 시장·플랫폼·가격·결제·광고 경로가 확인될 때만 본 실행안에 포함합니다.
- 한국에서 성립하지 않고 해외 소비자를 대상으로만 가능한 경우 `해외 타깃 확장안`으로 분리합니다.
- 미국 단가·전환율·구독료·제휴수수료를 한국 예상수익으로 그대로 옮기지 않습니다.
- 초보자가 오늘 바로 실행할 수 있도록 실제 제품명, 공식 링크, 오늘·3일·7일·30일 행동을 구체적으로 작성합니다.

각 원고에는 최소한 다음 흐름이 보여야 합니다.

`한국의 실제 고객 → 국내 유입 경로 → 판매·광고 경로 → 결제·정산 → 확정 수익 → 실제 입금`

이 흐름 중 핵심 단계가 확인되지 않으면 추정임을 밝히고, 총괄 편집실 확인사항으로 넘깁니다.

결과는 `handoff/draft/model-XX.md`에 저장합니다. 완료 후 `status/model-status.csv`의 해당 모델 `초안완료` 상태를 갱신합니다.
