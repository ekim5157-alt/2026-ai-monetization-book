# 모델 17. AI 업무자동화 구축 대행·컨설팅

- 상태: 초보자 실행 원고 v2
- 선행 문서: `handoff/business/model-17.md`, `handoff/research/model-17.md`
- 문장 형식 기준: `final/model-01.md`
- 공식 확인일: 2026-06-22

## 1. 한 줄 결론

AI 업무자동화 구축 대행·컨설팅은 **현재 업무 기준값 × 예외·사람 승인·복구 설계 × 최소 권한과 데이터 관리 × 사용량 원가 × 변경·유지보수 계약**이 맞아야 수익모델이 됩니다.

## 2. 이 모델은 무엇인가

영업·문의·문서·보고서·RAG·데이터 수집·커머스 연동·승인·업무 배분을 자동화하는 서비스입니다. 데모보다 현장 사용률, 예외 복구, 절감시간과 유지비가 중요합니다.

## 3. 성패를 가르는 것

월 처리 건수·건당 사람 시간·오류율·오류 1건의 손실을 먼저 측정합니다. 누락·중복·오타·긴급·권한 부족·외부 장애를 목록화하고 업무를 자동 실행·사람 승인·자동화 금지로 나눕니다.

중복 방지키, 입출력 로그, 오류 알림, 수동 전환, 재실행과 롤백을 설계합니다. 하자·기능 추가·업무 규칙 변경·외부 API 변경·긴급 대응을 계약에서 분리합니다.

## 4. 누가 돈을 지급하는가

고객 또는 프로젝트 플랫폼이 진단·PoC·본 구축·인수와 유지보수 계약에 따라 지급합니다.

## 5. 실행 경로

반복 업무 한 개의 실제 작업 20~50건을 관찰합니다. 정상 흐름·예외·승인·로그·복구를 포함한 진단서와 계약 조건을 만들고 작은 데이터로 시험운영합니다.

## 7. 주요 타겟

소상공인·쇼핑몰·교육·상담·제조·물류·기관·기업 실무팀처럼 반복 처리량과 오류비용이 있는 조직에 적합합니다.

## 8. 부가 타겟

한 업종의 예외·승인·복구 구조를 표준 패키지로 만들 수 있는 컨설턴트와 개발자에게 적합합니다.

## 9. 수익구조

유료 진단, PoC, 본 구축, 교육·인수인계, 월 유지보수와 긴급지원으로 나눕니다. 진단·개발·시험·API·서버·지원시간을 차감합니다.

## 10. 첫 수익 예상

유료 진단은 1~4주 안에 판매할 수 있습니다. 구축 매출은 진단과 계약 뒤 발생하고 반복수익은 유지보수에서 생깁니다.

## 11. 실제 입금 예상

`문의 → 진단 계약 → 설계·PoC → 현장 시험 → 검수·인수 → 잔금 → 유지보수` 순서로 기록합니다.

## 12. 필요한 제품과 공식 링크

OpenAI API — ChatGPT 구독과 별도 사용량 과금. 모델·검색·파일 비용과 데이터 보존 조건 확인. https://platform.openai.com/

CLOVA Studio — 한글 업무용 모델·도구 후보. 원화 단가는 최신 요금 화면 확인. https://www.ncloud.com/product/aiService/clovaStudio

Python·자체 서버 — 예외·로그·재처리·비용 한도 통제에 사용.

Google Sheets·GitHub — 기준 업무·예외표·버전·인수인계 관리. https://sheets.google.com/ · https://github.com/

## 13. 실행계획

### 오늘 실행

반복 업무 한 개의 월 처리량·시간·오류·자동·승인·금지 구간을 측정합니다.

### 3일 실행

정상 흐름·예외·승인·로그·복구를 포함한 진단서와 계약 조건을 만듭니다.

### 7일 실행

작은 데이터로 시험운영하고 비용·정확도·예외·복구시간을 기록합니다.

### 30일 실행

절감시간·오류·사용률·지원시간·고객가치·공급자 순이익을 비교합니다.

## 14. 자동화

입력 수집, 분류·요약 초안, 알림, 문서 생성과 상태 기록을 자동화할 수 있습니다. 고위험 판단과 예외 승인은 사람이 맡습니다.

## 15. 사람이 직접 할 일

사람은 업무 기준, 오류비용, 자동화 금지 구간, 데이터 흐름, 권한, 예외 책임자와 인수 기준을 결정합니다.

## 16. 실패 원인

현재 업무를 측정하지 않고 도구부터 연결하는 것, 정상 데이터만 시험하는 것, 사람이 전부 다시 확인하는 것, 외부 변경을 무상 하자로 보는 것이 대표적인 실패입니다.

## 17. 중단·전환 기준

현장 사용률이 낮거나 수동 복구가 원래 업무보다 오래 걸리고 지원시간이 계약금을 넘으면 범위와 가격을 바꿉니다.

## 18. 위험과 주의사항

API 학습 제외와 데이터 무보존은 다른 말입니다. 기능별 보존·삭제·외부 서비스 전송을 확인하고 비용 한도와 오류 로그를 둡니다.
