# Review handoff

전체 작업의 네 번째 단계입니다.

검증 전에 같은 모델의 다음 파일을 모두 읽습니다.

- `handoff/business/model-XX.md`
- `handoff/research/model-XX.md`
- `handoff/draft/model-XX.md`

검토 항목:
- 뻔한 내용만 남아 있지 않은가
- 실제 병목이 설명됐는가
- 조회수·클릭·예약·주문과 확정 수익·실제 입금을 구분했는가
- 공식 출처 없는 가격·자격·수수료를 단정했는가
- 약어와 용어설명이 중복되는가
- 오늘·3일·7일·30일이 실제 행동인가
- 한국 독자가 실행 가능한가
- 한국의 실제 구매자·독자·시청자·광고주가 특정돼 있는가
- 국내 유입·판매·광고·결제·정산 경로가 연결돼 있는가
- 한국 소비자가 납득할 가격과 구매 이유가 설명돼 있는가
- 플랫폼 수수료·광고비·환불·CS 비용을 제외해도 수익이 남는가
- 미국 사업자·콘텐츠 사업자의 단가·전환율·구독 구조를 한국에 그대로 옮겼는가
- 국내 경로가 없는 해외 모델을 한국용 실행안처럼 제시했는가

다음 중 하나라도 해당하면 치명적 오류로 분류합니다.

- 한국에서 누가 돈을 내는지 알 수 없음
- 실제 결제 또는 수익 지급 경로가 없음
- 미국 성공사례만 있고 한국 시장 근거가 없음
- 미국 광고단가·제휴수수료·구독료를 한국 예상수익으로 사용함
- 해외 타깃 모델과 국내 타깃 모델을 구분하지 않음

결과는 `handoff/review/model-XX.md`에 저장합니다. 완료 후 `status/model-status.csv`의 해당 모델 `검증완료` 상태를 갱신합니다.
