# 2026 AI 수익화 전체 지도

『2026 AI 수익화 전체 지도』 집필·검증을 위한 저장소입니다.

## 작업 흐름

`business → research → draft → review → final`

- `handoff/business/` — 사업성 분석과 조사 체크리스트
- `handoff/research/` — 공식자료 검증
- `handoff/draft/` — 초보자 실행 원고
- `handoff/review/` — 품질 검증
- `final/` — 승인 원고
- `status/` — 모델별 진행 상태
- `rules/` — 공통 편집 규칙
- `prompts/` — 단계별 작업 프롬프트

## 작업 원칙

- 한 번에 한 모델만 최종 편집합니다.
- 가격·자격·수수료·지급 조건은 공식 자료로 검증합니다.
- 조회수·클릭·문의·주문과 확정 수익·실제 입금을 구분합니다.
- 기존 파일을 수정할 때 같은 경로의 최신 파일을 갱신합니다.
- 작업 완료 후 `status/model-status.csv`를 갱신합니다.

## 파일명 규칙

```text
handoff/business/model-01.md
handoff/research/model-01.md
handoff/draft/model-01.md
handoff/review/model-01.md
final/model-01.md
status/model-status.csv
```

고정 모델명 중 5번은 `구글 SEO 블로그·애드센스(워드프레스)`, 12번은 `각종 어필리에이트`로 사용합니다.
